# Kontaktpersonen-Nachverfolgung
DH Projekt 5. Semester: Kontaktpersonen-Nachverfolgung 

## Technische Details
Umgebung/Plattform: React / ASP .NET Core

## Kurzzusammenfassung
Im Projekt soll ein Softwaresystem konzipiert und prototypisch implementiert werden,
das im Kern die Aufgaben unterstützt, wie sie bei der KontaktpersonenNachverfolgung durch die Gesundheitsämter anfallen (Identifikation von
Infektionsketten, Überblick über die aktuelle Gesundheitssituation, Erkennen von
regionalen Häufungen und Herdsituationen / Cluster). Denkbare Erweiterungen wären
zusätzliche Komponenten wie etwa die Möglichkeit der Erfassung und Meldung von
Kontaktdaten durch Restaurants etc., oder auch die Führung eines persönlichen
Kontakttagebuchs einzelner Personen - um nur ein paar Beispiele zu nennen.
